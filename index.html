JUniversal

Create cross platform mobile apps--in Java

A new approach to cross platform mobile

Why write the same app from scratch for every mobile platform? Want a way to take the tools & even codebase you use to develop for Java/Android today & use that to build cross platform apps that run on Windows and iOS too?

JUniversal uses source code translation, translating Java to C# and (coming soon) C++/Objective C++. Translated source is very user friendly, preserving comments and formatting.
